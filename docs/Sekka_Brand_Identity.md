## 6. اللغة والنبرة (Language & Tone)

| الخاصية | القيمة |
|---|---|
| **اللغة** | عربي — عامية مصرية |
| **الاتجاه** | RTL (من اليمين للشمال) |
| **النبرة** | ودية، بسيطة، زي ما بتكلم صاحبك |

### أمثلة على النبرة:

| فصحى (لا تستخدم) | عامية مصرية (استخدم) |
|---|---|
| في الانتظار | مستني |
| تم الاستلام | استلمت الشحنة |
| في الطريق | في السكة |
| تم التسليم | اتسلّم |
| فشل التسليم | معرفتش أسلّم |
| إعادة المحاولة | هجرب تاني |
| أدخل رقم الهاتف | اكتب رقم الموبايل |
| عنوان التسليم مطلوب | لازم تكتب عنوان التسليم |
| تأكيد التسليم | أكّد التسليم |
| إنشاء طلب جديد | أضف طلب |

### قواعد النصوص:
- **مفيش فصحى** في الواجهة — كل حاجة بالعامية المصرية
- الرسائل قصيرة ومباشرة
- استخدم "إنت" مش "أنتم"
- استخدم "عشان" مش "لكي" أو "من أجل"
